# CineTrack

A premium movie tracking application built with React and Tailwind CSS.

## Features

- **Dashboard**: View all your movies in a responsive grid.
- **Add Movie**: Add new movies to your watchlist.
- **Search & Filter**: Find movies instantly with real-time search and status filtering.
- **Local Storage**: Your data is saved locally on your device.
- **Animations**: Smooth transitions and hover effects using Framer Motion.
- **Dark Cinematic Theme**: A modern, sleek UI inspired by popular streaming platforms.

## Tech Stack

- React.js (Vite)
- Tailwind CSS
- Framer Motion
- React Icons
- React Hot Toast

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
